# Otimiza-ai Landing Page

Uma landing page high-tech moderna e responsiva para a empresa Otimiza-ai, especializada em soluções de automação com IA para varejo, beleza e alimentação.

## 🚀 Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **Tailwind CSS** - Framework de estilização utilitária
- **JavaScript Vanilla** - Animações e interatividade
- **Canvas API** - Sistema de partículas 3D
- **Intersection Observer API** - Animações de scroll
- **Font Awesome** - Ícones
- **Google Fonts** - Tipografia Inter

## ✨ Características Principais

### Design Visual
- **Estilo Deep Tech Minimalist** com elementos de Glassmorphism
- **Paleta de cores dark mode** com acentos neon (ciano, violeta, esmeralda)
- **Animações fluidas** e efeitos de hover interativos
- **Design responsivo** mobile-first

### Animações e Efeitos
- **Sistema de partículas 3D** no background do hero
- **Animações de scroll** com Intersection Observer
- **Efeitos hover neon** em botões e cards
- **Header dinâmico** com blur effect ao rolar
- **Glassmorphism** em cards e elementos

### Estrutura da Página
1. **Header** - Navegação com efeito backdrop blur
2. **Hero Section** - Título principal com partículas animadas
3. **Tech Intro** - Três pilares com ícones (IA Nativa, Integração API, Dados em Tempo Real)
4. **Solutions Section** - Três cards de produtos (Meritus, Otimiza Beauty, Otimiza Foods)
5. **Final CTA** - Chamada para ação principal
6. **Footer** - Informações de contato e links sociais

## 📱 Responsividade

A landing page é totalmente responsiva com breakpoints para:
- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px

### Ajustes Mobile:
- Menu hamburger para navegação
- Cards empilhados verticalmente
- Fontes ajustadas para melhor legibilidade
- Touch effects nos elementos interativos

## 🎯 Funcionalidades Implementadas

### Animações de Scroll
- Fade-in-up para cards e elementos
- Slide-in-left para ícones

- Header dinâmico com classes ao rolar

### Efeitos Neon
- Botões com glow effect ao hover
- Cards com bordas neon coloridas
- Transições suaves e escalas

### Interatividade
- Navegação suave entre seções

## 🎨 Personalização

### Cores
As cores principais podem ser ajustadas no CSS customizado:
```css
:root {
    --electric-cyan: #00f5ff;
    --deep-purple: #8b5cf6;
    --emerald-green: #10b981;
}
```

### Animações
Ajuste a velocidade e intensidade das animações em `js/animations.js`

### Partículas
Configure o sistema de partículas em `js/particles.js`:
- Número de partículas
- Velocidade de movimento
- Distância de conexão
- Cores disponíveis
